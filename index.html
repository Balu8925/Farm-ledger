How it works

A simple 3-step explanation (no blockchain jargon required).

1

A market agent submits a price

The agent connects a wallet and submits a price (commodity, amount, location, source). The wallet signs a transaction so the record is linked to the agent’s address.

2

The price is stored in the transaction note

We put the price as JSON text in the transaction’s note field (ARC-2 format). Once confirmed, it becomes a permanent, public record on Algorand TestNet.

3

Anyone can verify it later

Farmers can paste the TxID (or search by address) to fetch the transaction from the Indexer, decode the note, and see the exact data that was recorded on-chain.

Note: This demo uses TestNet. For real deployment, use MainNet and apply authorization rules (e.g., only approved market agents).
